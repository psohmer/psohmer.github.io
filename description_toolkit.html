UC Guidelines for Born-Digital Archival Description
Useful, practical guidelines for how to apply DACS-compliant archival description to born-digital records. Referenced widely. The guidelines do not provide on what direction on processing levels to apply to born-digital material, but do offer elaboration and examples of specific cases. Version 1.0 available on Github.
Tags: Born-Digital Description; Documentation
Encoded Archival Description (EAD) in Special Collections
Blog post on 2015 project at UMBC to create EAD-XML finding aids using PastPerfect. Includes link to the author’s tool to batch convert PastPerfect-XML to EAD-XML on Github. Also see UMBC Archives Processing Manual: Description, pages 8-9.
Tags: PastPerfect; Finding Aid; Institutional
Historical Society of Pennsylvania Guide to Using PastPerfect for Archival Description
Detailed guidelines from HSP for using PastPerfect to create description. Has field-by-field instructions and provides options for producing finding aids using PastPerfect.
Tags: PastPerfect; Documentation; Finding Aid; Institutional
NYU Archival Collections Management Program Documentation
Documentation on accessioning, arrangement and description from NYU. Useful as reference when considering holistic approach to program development and interoperability of archival functions, especially based around ArchiveSpace. Links to forms used by NYU - could be used as templates. List of Required Fields for description, connected to specific DACS rules, is helpful.
Tags: Documentation; Institutional

Documenting the Now White Paper
"Documenting The Now White Paper: Ethical Considerations for Archiving Social Media Content Generated by Contemporary Social Movements: Challenges, Opportunities, and Recommendations."" Provides examples of ethical challenges to collecting social media content and gives some recommendations about how to do so in a ethical way. "What is legal is not always ethical, and what is ethical is not always legal." See final page for four recommendations.
Tags: Born-Digital Description; Ethics
No one owes their trauma to archivists, or, the commodification of contemporaneous collecting
An excellent piece by Eira Tansey on “contemporaneous collecting” of archival materials during major events. She encourages critical thinking regarding pushes to document trauma, pointing out they are often done to appease GLAM administrators and that the people most affected by events (like COVID) are the most unable to participate. An excellent article to return to with some collected thoughts.
Tags: Blog; Ethics
Archives for Black Lives in Philadelphia Anti-Racist Description Resources
"Best practice recommendations for an anti-oppressive approach to creating and remediating archival description." Good to reference when writing potentially sensitive description. Includes lengthy annotated bibliography for additional sources on topic.
